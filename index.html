5. 정책 및 예외 처리

프린트 기능은 학부모 전달 목적이 포함되므로 개인정보 노출과 데이터 시점을 명확히 표시해야 함.

구분 정책 비고
학생명 표시 학급 분석/우리 반 리포트는 기본 마스킹, 개인별 리포트는 해당 학생명 표시 학부모 전달 시 타 학생 정보 노출 방지
데이터 없음 미제출/미학습 상태와 구분 필요

출력 방식 사용자가 프린트 또는 PDF 형식 저장 중 하나를 선택할 수 있도록 제공 PDF 형식 저장은 브라우저 인쇄의 PDF 저장과 별도 PDF 다운로드 방식으로 구현 가능
스크롤 영역 출력 화면 내 스크롤이 발생하는 목록, 표, 그래프 영역은 출력 시 스크롤을 제거하고 전체 데이터를 펼쳐 출력 학생 명단, 성취기준 목록, 답안 보기, 활동 결과 등 화면에서 일부만 보이는 데이터가 누락되지 않도록 처리
출력 페이지 분리 펼쳐진 데이터가 한 페이지를 초과할 경우 자동으로 다음 페이지에 이어서 출력 표의 행, 학생 카드, 성취기준 항목 등 하나의 데이터 단위가 페이지 중간에서 잘리지 않도록 분리 기준 적용
[참고] @media print, break-inside: avoid 활용
화면 UI와 출력 UI 차이 화면에서는 가독성을 위해 고정 높이와 내부 스크롤을 사용할 수 있으나, 출력/PDF에서는 데이터 누락 방지를 위해 출력 전용 레이아웃 사용 단순 화면 캡처 방식의 출력은 지양하며, 출력 시에는 스크롤 영역을 해제하고 전체 항목 표시
